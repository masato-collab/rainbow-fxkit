経済指標カウントダウン

EconomicCalendar_Countdown — 重要経済指標までの残り時間を常時可視化

概要

経済指標発表の直前直後は値動きが激しく、知らずにエントリーして飛ばされる事故が後を絶ちません。本インジケーターは次回の主要経済指標までのカウントダウンをチャート右上に常時表示し、指定時間前にアラートを鳴らします。重要度別に色分けされるので一目で判断できます。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 次回の重要経済指標までのカウントダウン表示(HH:MM:SS)
  • 重要度(★★★ / ★★ / ★)別の色分け表示
  • 指定時間前にポップアップ・サウンド・プッシュ通知でアラート
  • 表示対象通貨のフィルタリング(USD/EUR/JPY/GBP 等)
  • 設定ファイル(CSV)で指標スケジュールをユーザー編集可能
  • 表示コーナー・フォント・色をカスタマイズ可能

使い方

  1. zipを解凍しEconomicCalendar_Countdown.ex4と同梱CSVを取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Indicators に配置、CSVは MQL4 → Files に配置
  3. MT4を再起動してチャートへドラッグ
  4. 対象通貨・重要度のフィルタを設定
  5. 必要に応じてCSVを編集して最新の指標予定を反映

パラメーター

パラメーター名型デフォルト説明
ECC_CsvFileNamestring"calendar.csv"指標スケジュールのCSVファイル名
ECC_FilterCurrenciesstring"USD,EUR,JPY"表示対象通貨(カンマ区切り)
ECC_MinImportanceint2表示する最低重要度(1〜3)
ECC_AlertMinutesBeforeint15何分前にアラートを鳴らすか
ECC_UseAlertbooltrueポップアップアラートを有効化
ECC_UsePushboolfalseプッシュ通知を有効化
ECC_DisplayCornerenumRIGHT_UPPER表示位置
ECC_ColorHighcolorclrOrangeRed重要度★★★の色
ECC_ColorMidcolorclrGold重要度★★の色
ECC_ColorLowcolorclrSilver重要度★の色

注意事項

⚠️ 指標スケジュールCSVはユーザー自身で定期的に更新が必要です。予定変更や臨時発表には対応できません。重要な取引判断の前には必ず公式カレンダーでご確認ください。

このツールのFAQ

CSVファイルのフォーマットは?

同梱のcalendar_sample.csvをご参照ください。日時(UTC)、通貨、指標名、重要度(1-3)の4列です。

カウントダウンが表示されません

CSVがFilesフォルダにあるか、ECC_FilterCurrenciesが適切か、今後のイベントがCSVに含まれているかご確認ください。

指標データを自動取得できますか?

Web通信には対応していません。手動でCSVを更新する運用を想定しています。
