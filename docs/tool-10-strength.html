通貨強弱メーター

CurrencyStrength_Meter — 主要8通貨の強弱をランキング表示、最強×最弱ペアを自動提示

概要

「どの通貨ペアをトレードすべきか」の判断に役立つのが通貨強弱です。本インジケーターはUSD/EUR/JPY/GBP/AUD/NZD/CAD/CHFの主要8通貨の強弱を計算し、バーグラフでランキング表示。同時に「最強通貨×最弱通貨」の組み合わせを推奨ペアとして提示します。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 主要8通貨の強弱をバーグラフで可視化
  • 強さランキング(1位〜8位)を数値付きで表示
  • 最強通貨×最弱通貨のペアを自動算出して推奨表示
  • 計算期間と対象時間足をパラメーターで指定可能
  • 強弱の順位変動(クロスオーバー)時にアラート
  • バーの色をRainbow Projectのツールカラーに統一可能

使い方

  1. zipを解凍しCurrencyStrength_Meter.ex4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Indicators に配置
  3. MT4を再起動しチャートにドラッグ
  4. 計算する時間足(デフォルトH1)と期間を設定
  5. 推奨ペアをチャートで確認し、他のツールと併用してエントリー検討

パラメーター

パラメーター名型デフォルト説明
CSM_TimeframeenumPERIOD_H1計算に使う時間足
CSM_Periodint24強弱計算に使う本数(デフォルト24本=1日分)
CSM_UpdateIntervalint60強弱の再計算間隔(秒)
CSM_ShowRecommendPairbooltrue最強×最弱の推奨ペアを表示
CSM_AlertOnRankChangebooltrue順位変動時にアラート
CSM_PanelCornerenumLEFT_UPPERパネル表示位置
CSM_BarWidthint180バーグラフの幅(px)

注意事項

⚠️ 強弱計算は「全通貨の関連ペア(ドルストレート・クロス)」のデータを使うため、気配値表示で主要ペアを全て有効化しておく必要があります。データが不足するとバーが空白になります。

このツールのFAQ

バーが全く表示されません

MT4「気配値表示」ウィンドウで、USD/EUR/JPY/GBP/AUD/NZD/CAD/CHFの全組み合わせ(28ペア)を「全て表示」にしてください。

ブローカーの通貨ペア表記が違います

EURUSD.fx など接尾辞が付くブローカーもあります。パラメーターCSM_SymbolSuffixで指定可能です。

どの時間足がおすすめ?

デイトレードならH1×24本(1日分)、スイングならH4×30本(約5日分)が目安です。
