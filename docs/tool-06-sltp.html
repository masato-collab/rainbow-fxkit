#06 · UTILITY EA

ビジュアル損切り・利確ライン

VisualSLTP_Tool — SL/TPラインをドラッグで直感調整、損益が即可視化

概要

エントリー・SL・TPラインをチャート上にドラッグ&ドロップで配置できるビジュアルツールです。線を動かすごとに、pips数・損益額(円換算)・リスクリワード比率がリアルタイム更新されるため、「今このラインだと損失いくらか」を数字で把握しながらプラニングできます。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • エントリー / SL / TPの3本のラインをチャート上にドラッグ配置
  • ライン移動ごとにpips数・損益額・RR比をリアルタイム表示
  • 想定ロットを入力すると損益額が円・USDで試算される
  • エントリー方向(買い/売り)の切替ボタン
  • 現在のSL/TP設定を既存ポジションに一括反映するボタン
  • 塗りつぶしで利益エリア(緑)/損失エリア(赤)を視覚化

使い方

  1. zipを解凍しVisualSLTP_Tool.ex4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Experts に配置
  3. 自動売買を有効化してチャートにドラッグ
  4. 3本のラインをドラッグしてSL/TPを決定
  5. 「既存ポジションに適用」ボタンで実際のSL/TPに反映

パラメーター

パラメーター名型デフォルト説明
VST_DefaultLotsdouble0.10試算に使う初期ロット
VST_DirectionenumBUY起動時のエントリー方向
VST_EntryColorcolorclrYellowエントリーラインの色
VST_SLColorcolorclrTomatoSLラインの色
VST_TPColorcolorclrMediumSeaGreenTPラインの色
VST_FillProfitAreabooltrue利益/損失エリアを塗りつぶす
VST_AutoApplyToPositionsboolfalse線移動で既存ポジションに自動反映(確認ダイアログ付き)

注意事項

⚠️ 「既存ポジションに適用」ボタンは実際のSL/TP変更処理を行います。意図しないポジションに反映されないよう、通貨ペア・マジックナンバーを事前に確認してください。

このツールのFAQ

ラインが表示されません

自動売買が有効になっているかご確認ください。また、パラメーターVST_Directionに応じて3本のラインがチャート中央付近に生成されます。

RR比が計算されません

SLとエントリー間の距離が0の場合は計算できません。SLをエントリーから少し離してください。

複数ポジションに一括適用したい

マジックナンバーフィルタを指定すれば、該当するポジションだけに一括反映可能です。
