相関通貨ペアモニター

CorrelationMatrix_Monitor — 主要通貨ペアの相関をリアルタイムで可視化

概要

相関の高いペアを同方向に複数ポジション取ることは、実質的にロットを倍増させているのと同じで、一気に資金を失う典型的パターンです。本インジケーターは主要通貨ペアの相関係数をマトリクス表示し、通常と異なる動き(ダイバージェンス)が発生したときにアラートを鳴らします。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 指定した主要通貨ペア間の相関係数をマトリクス表示(-1.0 〜 +1.0)
  • 相関の強弱に応じて色分け(正相関=緑、逆相関=赤)
  • 計算期間(短期/長期)を切り替え可能
  • 通常の相関関係が崩れたダイバージェンス検出とアラート
  • リスク分散の観点で「同方向エントリー禁止ペア」をハイライト

使い方

  1. zipを解凍しCorrelationMatrix_Monitor.ex4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Indicators に配置
  3. MT4を再起動しチャートにドラッグ
  4. 対象通貨ペアと計算期間を設定
  5. マトリクスが表示されたらダイバージェンスアラートを有効化

パラメーター

パラメーター名型デフォルト説明
CMM_Pairsstring"EURUSD,GBPUSD,USDJPY,AUDUSD,USDCHF,USDCAD"監視対象ペア(カンマ区切り)
CMM_Periodint100相関計算の期間(本数)
CMM_TimeframeenumPERIOD_H1計算に使う時間足
CMM_AlertThresholddouble0.3相関ずれがこの値を超えたらアラート
CMM_UseAlertbooltrueダイバージェンスアラートを有効化
CMM_DisplayCornerenumLEFT_UPPERマトリクスの表示位置
CMM_ColorPositivecolorclrLimeGreen正相関時の色
CMM_ColorNegativecolorclrTomato逆相関時の色

注意事項

⚠️ 相関関係は時間とともに変化します。過去のデータに基づく相関が現在や将来もそのまま続くとは限りません。あくまでリスク判断の一助としてご活用ください。

このツールのFAQ

マトリクスの数値が更新されません

該当する通貨ペアのチャートを一度表示してヒストリーデータを取得する必要があります。気配値表示で該当ペアを「全て表示」してください。

計算が重いです

CMM_Periodを小さくするか、監視ペア数を減らすと軽くなります。

ダイバージェンスアラートはどう使う?

通常の相関関係が崩れている時は、いずれか一方が先に動いている合図です。トレンド転換や個別要因のヒントに使えます。
