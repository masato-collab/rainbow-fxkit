複数時間足トレンド一覧

MultiTF_TrendPanel — 全時間足のトレンドを信号機で一目把握

概要

短期の動きだけを見て逆張りしてしまい、大きな流れに飲まれる——これもよくあるパターンです。本インジケーターはM5/M15/M30/H1/H4/D1/W1の複数時間足のトレンド方向を同時に計算し、上昇=緑/下降=赤/レンジ=灰色の信号機パネルで一覧表示します。全足一致時にアラートを鳴らす機能も。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

スクリーンショット(後日差し替え)

主な機能

  • M5〜W1の最大7時間足を一覧表示
  • 上昇(緑)/下降(赤)/レンジ(灰)の信号機形式
  • トレンド判定方式を3種(MA/ADX/高安更新)から選択可能
  • 全時間足の一致度(0〜100%)を算出
  • 強いトレンド発生時(一致度が閾値超え)にアラート
  • 監視する時間足を個別にオン/オフ可能

使い方

  1. zipを解凍しMultiTF_TrendPanel.mq4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Indicators に配置
  3. MT4を再起動しチャートにドラッグ
  4. トレンド判定方式と有効化する時間足を選択
  5. 一致度アラートの閾値(例:80%)を設定

パラメーター

パラメーター名型デフォルト説明
MTP_MethodenumMETHOD_MAトレンド判定方式(MA/ADX/HL)
MTP_MAPeriodint50MA方式で使うMAの期間
MTP_ADXThresholddouble25.0ADX方式でトレンドと判定する閾値
MTP_EnableM5〜MTP_EnableW1booltrue各時間足の表示有効化
MTP_AlertMatchPctdouble80.0一致度がこの%を超えたらアラート
MTP_UseAlertbooltrue一致度アラートを有効化
MTP_PanelCornerenumRIGHT_UPPERパネル表示位置

注意事項

⚠️ トレンド方向の判定は、使用する判定方式によって結果が異なります。同じチャートを異なる方式で見比べると、相場認識の幅が広がります。あくまで判断補助です。

このツールのFAQ

一致度が常に低く、アラートが出ません

レンジ相場では一致度が低くなりがちです。MTP_AlertMatchPctを70%程度まで下げるか、MTP_Methodを変えてみてください。

上位足だけを重視したい

M5/M15/M30をオフにして、H1以上のみ監視する構成にできます。

別の通貨ペアのトレンドも見られますか?

現状は表示中のチャートの通貨ペアのみです。別ペアは別チャートに当ててご利用ください。
