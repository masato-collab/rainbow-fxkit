ロット計算機

LotCalculator_Panel — 許容リスクから最適ロットを自動算出するチャート上パネル

概要

「今回の取引でロットはいくつが適切か」という根本的な問いに、チャート上のパネルで即答してくれるツールです。口座残高・リスク許容率(%)・SL幅(pips)を入力すると、最適ロットサイズ・想定損失額・RR別の想定利益額をリアルタイム計算します。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

スクリーンショット(後日差し替え)

主な機能

  • チャート上パネルに残高・リスク%・SL幅を入力
  • 最適ロットサイズを自動計算(通貨ペアごとのpip価値を考慮)
  • 想定損失額を円・USD両方で表示
  • RR 1:1, 1:2, 1:3 別の想定利益額を同時表示
  • ブローカー許可範囲(最小/最大ロット・ステップ)に自動クリップ
  • 結果をクリップボードにコピーするボタン付き

使い方

  1. zipを解凍しLotCalculator_Panel.mq4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Experts に配置
  3. MT4「ツール→オプション→エキスパートアドバイザ」で自動売買を有効化
  4. チャートにLotCalculator_Panelをドラッグ&ドロップ
  5. パネルでリスク%・SL幅を入力すると即座にロットが算出される

パラメーター

パラメーター名型デフォルト説明
LCP_DefaultRiskPctdouble2.0初期リスク許容率(%)
LCP_DefaultSLPipsdouble20.0初期SL幅(pips)
LCP_BaseOnEquitybooltrue計算ベースを残高(Balance)でなく有効証拠金(Equity)にする
LCP_RoundingModeenumROUND_DOWNロット丸め方向(切捨/四捨五入)
LCP_PanelCornerenumLEFT_UPPERパネルの表示位置
LCP_ShowJPYEquivalentbooltrue円換算の損益も表示
LCP_ShowRRAnalysisbooltrueRR 1:1〜1:3の利益額を表示

注意事項

⚠️ 本ツールは発注機能を持ちません。計算結果をもとに実際の発注は手動で行ってください。スプレッド・スリッページは計算に含まれないため、余裕を持った設定を推奨します。

このツールのFAQ

ドル円以外のクロス円でも正確ですか?

現在保有通貨と口座通貨のレートを考慮して計算しています。JPYペア・その他ペア両対応です。

ロットが最小ロット未満になります

リスクが小さすぎるかSL幅が広すぎます。LCP_RoundingModeをROUND_UPにすると最小ロットに繰り上げます。

パネルが表示されません

MT4の「自動売買」ボタンが有効になっているかご確認ください。右上のニコちゃんマーク(スマイル)が青色になっている必要があります。
