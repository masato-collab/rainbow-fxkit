キリ番アラート

RoundNumber_Alert — キリの良い価格帯への接近・到達を音と通知で逃さない

概要

FXでは X.X000 や X.X500 といった「キリ番」が強力なサポート/レジスタンスとして機能することが多く、見落とすと痛恨の損失につながります。本インジケーターはキリ番にラインを自動描画し、接近時と到達時に音・ポップアップ・メール/プッシュで通知します。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 指定pips間隔のキリ番を自動計算(10/25/50/100pipsなど)
  • チャート上下に指定本数分のキリ番水平ラインを自動描画
  • 残り○pipsでの接近アラート(距離はパラメーターで指定)
  • タッチ/ブレイク時の到達アラート
  • 同一キリ番への重複アラートを自動抑制(価格が離れるまで再発動しない)
  • JPYペア/その他ペアのpip単位を自動判定(3桁/5桁ブローカー対応)
  • 音・ポップアップ・メール・プッシュ通知すべて対応

使い方

  1. zipを解凍し RoundNumber_Alert.mq4 を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Indicators に配置
  3. MT4を再起動、ナビゲーターからチャートにドラッグ
  4. キリ番間隔(RNA_IntervalPips)と接近距離(RNA_ApproachPips)を設定
  5. 通知したい方法のフラグ(Alert/Sound/Mail/Push)をオン

パラメーター

パラメーター名型デフォルト説明
RNA_IntervalPipsdouble50.0キリ番の間隔(pips)。10/25/50/100 など
RNA_ApproachPipsdouble5.0接近アラートの発動距離(pips)
RNA_LinesAboveint5現在価格より上方向に描画するキリ番の本数
RNA_LinesBelowint5現在価格より下方向に描画するキリ番の本数
RNA_UseAlertbooltrueポップアップアラートを鳴らす
RNA_UseSoundbooltrue追加のサウンドを鳴らす
RNA_UseMailboolfalseメール通知を送信
RNA_UsePushboolfalseモバイルプッシュ通知を送信
RNA_LineColorcolorclrDodgerBlueキリ番ラインの色
RNA_LineWidthint1ラインの太さ
RNA_ResetBufferPipsdouble2.0アラートリセット用の追加バッファ(pips)

注意事項

⚠️ メール通知は MT4「ツール→オプション→メール」でSMTP設定、プッシュ通知は「通知機能」でMetaQuotes IDの登録が必要です。

このツールのFAQ

同じキリ番で何度も鳴ってしまいます

RNA_ResetBufferPipsを大きめ(3.0〜5.0)に設定してください。

ラインが描画されません

RNA_IntervalPipsが大きすぎる可能性があります。小さい値でお試しください。

サウンドが鳴りません

RNA_SoundApproach/RNA_SoundTouchに指定したWAVファイルがMT4のSoundsフォルダにあるかご確認ください。
