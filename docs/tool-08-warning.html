#08 · EA

自動ポジションサイズ警告

PositionSize_Warning — 一撃退場を防ぐための、3段階の守り神

概要

初心者の「一撃退場」の多くは、残高に対して過大なロット発注・SL未設定・高レバレッジの組み合わせで発生します。本EAは新規ポジション検知時にリスク額・レバレッジを自動計算し、基準を超える場合に3段階(注意/警告/危険)でチャート上とポップアップに警告を表示します。CSVログも残します。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 新規ポジション発注を自動検知
  • 残高に対するリスク額(SL距離 × ロット)を計算
  • 3段階(2%/5%/10%)のしきい値で警告を色分け(黄/橙/赤)
  • SL未設定ポジション検知時にも警告
  • 口座全体の有効レバレッジが上限超過時に警告
  • 全ての警告をCSVログファイルに自動記録
  • 警告はチャート上テキスト+ポップアップの二重表示

使い方

  1. zipを解凍しPositionSize_Warning.ex4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Experts に配置
  3. 自動売買を有効化して、監視したいチャート(任意のペアでOK)にドラッグ
  4. しきい値(2%/5%/10%)とレバレッジ上限を確認
  5. 以後、新規発注するたびに自動で警告判定が行われる

パラメーター

パラメーター名型デフォルト説明
PSW_Level1_Pctdouble2.0レベル1(注意/黄):残高の何%超えたら警告
PSW_Level2_Pctdouble5.0レベル2(警告/橙):残高の何%超えたら警告
PSW_Level3_Pctdouble10.0レベル3(危険/赤):残高の何%超えたら警告
PSW_MaxLeveragedouble10.0口座全体の有効レバレッジ上限
PSW_TextDurationint30チャート上の警告テキスト表示秒数
PSW_TextFontSizeint20警告テキストのフォントサイズ
PSW_LogFileNamestring"PositionSize_Warning_Log.csv"ログファイル名

注意事項

⚠️ SL未設定を「想定損失無限大」と見なすため必ず警告が出ます。SL未設定でもトレードしたい場合は警告を無視しても実害はありません(ただし資金管理上は推奨できません)。

このツールのFAQ

複数チャートで同時に使っても大丈夫?

1つのチャートにだけ適用してください。複数にアタッチすると警告が重複します。

CSVログはどこに出力されますか?

MT4データフォルダの MQL4/Files/ 以下に出力されます。「ファイル→データフォルダを開く」から辿れます。

他EAからの発注も警告対象?

はい、マジックナンバーに関係なく全てのポジションを監視します。
