#07 · UTILITY EA

ワンクリック全決済

OneClick_CloseAll — 複数の決済オプションをボタン一発で実行、緊急時の冷静な操作を支援

概要

複数ポジション保有中にパニックになり、個別決済を繰り返している間に相場がさらに動いて損失を拡大させる——これは多くの初心者が経験する悲劇です。本ツールはチャート上にボタンパネルを表示し、全決済・買いのみ・売りのみ・建値以上/以下決済などをワンクリックで実行できるようにします。

📥 ダウンロード (.zip) ← ツール一覧へ戻る

主な機能

  • 全決済ボタン: 口座内の全ポジションを一括決済
  • 買いのみ/売りのみ: 片方向だけを決済
  • 建値以上決済: 含み益のポジションのみ決済(利益確保)
  • 建値以下決済: 含み損のポジションのみ決済(損切り切替)
  • 同一通貨ペアのみ/全ペア対象の切替
  • 実行前に「本当に決済しますか?」確認ダイアログ
  • マジックナンバーフィルタで特定EA由来のポジションだけ決済可能

使い方

  1. zipを解凍しOneClick_CloseAll.ex4を取り出す
  2. MT4「データフォルダを開く」→ MQL4 → Experts に配置
  3. 自動売買を有効化してチャートにドラッグ
  4. パネルのボタンで決済種別を選択
  5. 確認ダイアログで「はい」を押すと決済実行

パラメーター

パラメーター名型デフォルト説明
OCA_CurrentSymbolOnlybooltrue現在のチャートの通貨ペアのみ対象にする
OCA_FilterMagicint0対象マジックナンバー(0=全て)
OCA_ConfirmBeforeClosebooltrue決済前に確認ダイアログを表示
OCA_PanelCornerenumRIGHT_LOWERパネルの表示位置
OCA_ButtonSizeint120ボタン幅(px)
OCA_Slippageint10決済時の許容スリッページ(point)
OCA_MaxRetryint3決済失敗時の再試行回数

注意事項

⚠️ 本ツールは実際の決済処理を行います。誤操作防止のため確認ダイアログをオンにしておくことを強く推奨します。大荒れ相場ではスプレッド拡大で想定外の価格で約定する可能性がある点にもご注意ください。

このツールのFAQ

「取引が許可されていません」エラーが出ます

MT4の「自動売買」ボタンを有効化し、ツールのプロパティ「全般」タブで「自動売買を許可」のチェックが入っているかご確認ください。

一部のポジションが決済されません

OCA_CurrentSymbolOnlyがtrueだと他ペアは対象外です。全ペア決済したい場合はfalseに。また、再試行しても失敗した場合はエキスパートログをご確認ください。

ボタンパネルをチャート外に追い出したい

パネルは現在のチャート内にのみ表示できます。フローティングは非対応です。
